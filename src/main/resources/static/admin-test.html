ADMIN CONSOLE

공연장부터 회차 인벤토리까지 한 번에 검증하기

이 페이지는 venue -> seats/sections -> event/show -> show inventory 흐름을 끊기지 않게 확인하기 위한 어드민 테스트 콘솔입니다.

사전 준비

  1. 먼저 ADMIN Access Token을 준비해 아래 입력창에 저장합니다.
  2. 공연장을 upsert 한 뒤 좌석/구역 기준정보를 등록합니다.
  3. 등록 후 조회 버튼으로 seatId / sectionId를 확인합니다.
  4. 이벤트와 회차를 만든 다음 회차 인벤토리를 생성합니다.
  5. 마지막에 가용성/홀드 페이지 열기로 USER 선점 흐름까지 이어서 확인합니다.
